## Stage 1: Setup

Good setup — self-selection into training is a textbook selection-on-observables problem. Let me confirm the frame before writing code:

> You're estimating the effect of a training program on productivity, using Python, with age, tenure, and department as covariates. Participants chose whether to enroll.

**Two quick questions before I proceed:**

1. **Estimand**: Do you want the **ATT** (effect on those who actually took the training — "did the program help the people who chose it?") or the **ATE** (effect on everyone, including those who didn't bother to enroll)? For voluntary programs, **ATT is usually the right target** — it tells you whether the program worked for its actual participants.

2. **Unobserved confounders**: Can you think of any variable that both drives who self-selects *and* affects productivity, that isn't in your data? For example: motivation, prior performance, manager encouragement, role complexity? If yes, that's a threat we'll need to address explicitly.

## Stage 2: Assumptions

Here's where matching stands or falls. Let me walk through each one for your setting:

---

### 1. 🔴 Conditional Independence (Unconfoundedness) — *Untestable, most critical*

**What it means here**: After conditioning on age, tenure, and department, the decision to enroll in training is unrelated to what productivity would have been without training. In other words: among people with the same age, tenure, and department, the ones who enrolled would have had the same *baseline productivity trajectory* as those who didn't.

**Why it might fail**: Self-selection is the enemy here. People who sign up for training likely differ from those who don't in ways your data doesn't capture — motivation, ambition, a supportive manager, awareness of a promotion opportunity. Those unmeasured drivers also affect productivity. If that's true, your estimated effect conflates "training helped" with "motivated people are more productive."

**Assessment**: With only 3 covariates, this is a significant concern. I'd rate unconfoundedness as *moderately implausible* unless you have strong domain knowledge that age, tenure, and department fully explain enrollment patterns.

**What we'll do**: Run a sensitivity analysis (E-value) later to quantify how strong an unobserved confounder would need to be to explain away the result.

---

### 2. ✅ Overlap / Positivity — *Testable*

**What it means**: Within every combination of (age, tenure, department), there are both treated and untreated employees. No demographic profile that only ever enrolls or never enrolls.

**Why it matters**: If all senior employees enrolled and no junior ones did, there's nothing to compare — matching would have to extrapolate, producing unreliable estimates.

**Testable** — the first thing we'll run is an overlap diagnostic.

---

### 3. ⚠️ SUTVA (No Interference)

**What it means**: One employee's training doesn't affect another's productivity. No peer learning spillovers, no competitive effects within teams.

**Assessment**: Plausible if employees work independently. Could be violated if trained employees share skills with teammates or if there's within-team competition. Flag this if your organization is highly collaborative.

---

### 4. ⚠️ Correct Model Specification

**What it means**: The propensity score model (logistic regression predicting enrollment) needs to adequately capture who selects into training. With only 3 covariates, you're relying heavily on this model being right.

**Testable**: We'll check covariate balance (standardized mean differences) after matching/weighting. If balance is poor, the model is misspecified.

**Plan**: We'll use doubly-robust estimation (DR-Learner) as a safeguard — if either the propensity model or the outcome model is correct, we get a consistent estimate.

# ── 0. Load data ──────────────────────────────────────────────
# Replace with: df = pd.read_csv("your_data.csv")
df = pd.DataFrame(...)  # your dataframe here

TREATMENT   = "treated"

COVARIATES  = ["age", "tenure", "department"]

Y = df[OUTCOME].values
D = df[TREATMENT].values

scaler = StandardScaler()

ps_model.fit(X_scaled, D)

axes[0].hist(ps[D == 1], bins=30, alpha=0.6, color="steelblue", label="Treated", density=True)

print(f"Extreme PS > 0.95: {(ps > 0.95).sum()} units")

# ── 2. Covariate Balance: Before Weighting ───────────────────
def compute_smd(df, covariates, treatment, weights=None):
    """Standardized Mean Differences for each covariate."""
    rows = []
    for cov in covariates:
        treated_vals  = df.loc[df[treatment] == 1, cov].values
        control_vals  = df.loc[df[treatment] == 0, cov].values
        pooled_std = np.sqrt(
            (treated_vals.var() + control_vals.var()) / 2
        )
        smd_raw = (treated_vals.mean() - control_vals.mean()) / pooled_std

        if weights is not None:
            w = weights[df[treatment] == 0]
            weighted_mean_c = np.average(control_vals, weights=w)
            smd_w = (treated_vals.mean() - weighted_mean_c) / pooled_std
        else:
            smd_w = None

        rows.append({"covariate": cov, "SMD_before": smd_raw, "SMD_after": smd_w})
    return pd.DataFrame(rows)

# ── 3. IPW Weights (ATT) ──────────────────────────────────────
# ATT weight: 1 for treated, ps/(1-ps) for controls
df["ipw"] = np.where(D == 1, 1.0, ps / (1 - ps))

# Stabilized weights to reduce variance
df["ipw_stabilized"] = np.where(
    D == 1,
    1.0,
    (ps / (1 - ps)) * ((1 - ps).mean() / ps.mean())
)

# Check weight distribution
print("\n" + "=" * 50)
print("IPW WEIGHT DIAGNOSTICS (stabilized, control only)")
print("=" * 50)
w_ctrl = df.loc[D == 0, "ipw_stabilized"]
print(f"  Mean: {w_ctrl.mean():.3f}  |  Max: {w_ctrl.max():.3f}  |  "
      f"P99: {np.percentile(w_ctrl, 99):.3f}")
print("  (Extreme max weight = overlap problem; consider trimming)")

# ── 4. Balance After IPW ──────────────────────────────────────
balance = compute_smd(df, COVARIATES, TREATMENT, weights=df["ipw_stabilized"].values)
print("\n" + "=" * 50)
print("COVARIATE BALANCE (SMD)")
print("=" * 50)
print(balance.to_string(index=False))
print("\n  Rule of thumb: |SMD| < 0.10 after weighting = good balance")
print(f"  All balanced: {(balance['SMD_after'].abs() < 0.10).all()}")

# Love plot
fig, ax = plt.subplots(figsize=(8, 4))
y_pos = range(len(COVARIATES))
ax.scatter(balance["SMD_before"].abs(), y_pos, marker="o", color="salmon",
           s=80, label="Before weighting", zorder=3)
ax.scatter(balance["SMD_after"].abs(),  y_pos, marker="D", color="steelblue",
           s=80, label="After IPW",       zorder=3)
ax.axvline(0.10, color="gray",  linestyle="--", alpha=0.7, label="|SMD| = 0.10")
ax.axvline(0.25, color="black", linestyle=":",  alpha=0.5, label="|SMD| = 0.25")
ax.set_yticks(y_pos)
ax.set_yticklabels(COVARIATES)
ax.set_xlabel("Absolute Standardized Mean Difference")
ax.set_title("Love Plot: Covariate Balance Before vs. After IPW")
ax.legend()
plt.tight_layout()
plt.savefig("love_plot.png", dpi=150)
plt.show()

# ── 5. Primary Estimate: Doubly-Robust (ATT) ──────────────────
# Uses cross-fitting to avoid overfitting bias
from econml.dr import DRLearner
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler

dr = DRLearner(
    model_propensity = GradientBoostingClassifier(n_estimators=100, random_state=42),
    model_regression = GradientBoostingRegressor(n_estimators=100, random_state=42),
    model_final      = LinearRegression(),
    cv               = 3,

dr.fit(Y=Y, T=D, X=X, W=None)

# ATT estimate (effect on the treated)
X_treated = X[D == 1]
ate_dr  = dr.ate(X_treated)
ate_ci  = dr.ate_interval(X_treated, alpha=0.05)

print("\n" + "=" * 50)
print("PRIMARY ESTIMATE: Doubly-Robust (ATT)")
print("=" * 50)
print(f"  Point estimate : {ate_dr:.4f}")

model_ipw = smf.wls(

# ── 8. Sensitivity Analysis: E-value ──────────────────────────
# How strong would an unobserved confounder need to be to
# explain away the DR estimate?
# E-value formula for a mean difference (linear outcome):
#   Convert effect to risk ratio equivalent, then compute E-value.

def e_value_linear(estimate, se, outcome_sd):
    """
    Approximates E-value for a standardized mean difference.
    RR approximation: exp(0.91 * d) where d = estimate / outcome_sd
    """
    d  = abs(estimate) / outcome_sd
    rr = np.exp(0.91 * d)
    ev = rr + np.sqrt(rr * (rr - 1))
    # E-value for CI bound
    d_lo  = max(0, abs(estimate) - 1.96 * se) / outcome_sd
    rr_lo = np.exp(0.91 * d_lo)
    ev_lo = rr_lo + np.sqrt(rr_lo * (rr_lo - 1))
    return ev, ev_lo

outcome_sd = df["productivity"].std()
# Use IPW estimate + SE for E-value (DR SE requires bootstrap)
ev, ev_lo = e_value_linear(
    estimate   = ate_ipw,
    se         = model_ipw.bse["treated"],
    outcome_sd = outcome_sd
)
print("\n" + "=" * 50)
print("SENSITIVITY ANALYSIS: E-value")
print("=" * 50)
print(f"  E-value (point estimate): {ev:.2f}")
print(f"  E-value (CI lower bound): {ev_lo:.2f}")
print(
    f"\n  Interpretation: An unobserved confounder would need a\n"
    f"  risk-ratio-scale association of ≥ {ev:.2f} with BOTH\n"
    f"  treatment selection and productivity to fully explain\n"
    f"  away the observed effect. The CI bound E-value of {ev_lo:.2f}\n"
    f"  is the more conservative threshold."
)

# ── 9. Placebo Outcome Test ────────────────────────────────────
# Test on a pre-treatment proxy or unrelated variable.
# Here we use 'age' as a pseudo-outcome — training can't change age.
# A significant 'effect' on age would indicate residual confounding.

model_placebo = smf.wls(
    "age ~ treated + tenure + C(department)",
    data=df,
    weights=df["ipw_stabilized"]
).fit(cov_type="HC3")

ate_placebo = model_placebo.params["treated"]
p_placebo   = model_placebo.pvalues["treated"]
print("\n" + "=" * 50)
print("PLACEBO TEST: 'Effect' on Age (should be ~0)")
print("=" * 50)
print(f"  Estimate: {ate_placebo:.4f}  |  p-value: {p_placebo:.4f}")
if p_placebo < 0.05:
    print("  ⚠️  WARNING: Significant placebo effect — residual confounding likely.")
else:
    print("  ✅ No significant placebo effect — balance looks adequate.")

# ── 10. Propensity Score Trimming Robustness ──────────────────
# Re-run on the trimmed sample (drop extreme PS)
TRIM_LO, TRIM_HI = 0.10, 0.90
df_trim = df[(df["ps"] >= TRIM_LO) & (df["ps"] <= TRIM_HI)].copy()

model_trim = smf.wls(
    "productivity ~ treated + age + tenure + C(department)",
    data=df_trim,
    weights=df_trim["ipw_stabilized"]
).fit(cov_type="HC3")

ate_trim = model_trim.params["treated"]
ci_trim  = model_trim.conf_int().loc["treated"]
print("\n" + "=" * 50)
print(f"TRIMMING ROBUSTNESS (PS in [{TRIM_LO}, {TRIM_HI}], n={len(df_trim)})")
print("=" * 50)
print(f"  Trimmed estimate: {ate_trim:.4f}  |  95% CI: [{ci_trim[0]:.4f}, {ci_trim[1]:.4f}]")
print(f"  Full estimate   : {ate_ipw:.4f}")
delta = abs(ate_trim - ate_ipw)
print(f"  Change          : {delta:.4f}  {'⚠️ Large change — overlap is fragile' if delta > 1.0 else '✅ Stable'}")
