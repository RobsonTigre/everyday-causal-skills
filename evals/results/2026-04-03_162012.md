## Stage 2: Assumptions

Let's walk through each one for your context.

### 1. No concurrent confounding events ❓ (untestable — must be argued)

> **Why it matters**: ITS can only attribute the post-period change to the policy if *nothing else* changed at month 37. If a second policy, economic shock, or seasonal anomaly coincided with your intervention, the estimated effect is a mix of both — and there's no statistical way to separate them.

**Your action**: Think carefully — did anything else happen around month 37 that might have affected your outcome? A second policy, a data collection change, a market shock?

### 2. Pre-period model fit ✅ (testable — code below)

> **Why it matters**: The entire counterfactual logic rests on projecting the pre-period trend forward. If the linear trend model fits the pre-period poorly (high MAPE, autocorrelated residuals), the projected counterfactual is unreliable — you're extrapolating a bad model.

### 3. Stationarity ✅ (testable — code below)

> **Why it matters**: If your series has a unit root (random walk), apparent post-intervention changes can just be noise accumulation — not a real effect. The ADF and KPSS tests check this directly.

### 4. Adequate pre-period length ⚠️

> **Why it matters**: With only 36 pre-period observations you're at the minimum. If there's annual seasonality, you have exactly 3 cycles — enough to detect it but not abundantly. The model may underfit the seasonal pattern, widening your confidence intervals and potentially biasing the slope estimate.

## Stage 3: Implementation

```python
# ── Dependencies ──────────────────────────────────────────────────────────────
# pip install statsmodels scipy matplotlib pandas numpy

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import statsmodels.formula.api as smf
from statsmodels.stats.diagnostic import acorr_ljungbox
from statsmodels.tsa.stattools import adfuller, kpss
from scipy.stats import shapiro

# ── 1. Load your data ─────────────────────────────────────────────────────────
# Replace this with pd.read_csv(...) for your actual file
# df = pd.read_csv("your_data.csv")

INTERVENTION_TIME = 36   # last pre-period time point (intervention starts at 37)

# ── 2. Feature engineering ────────────────────────────────────────────────────
df["time_since"] = np.maximum(0, df["time"] - INTERVENTION_TIME)

# Confirm structure
print(df[["time", "outcome", "post", "time_since"]].head(10))
print(f"\nPre-period obs : {(df['post'] == 0).sum()}")
print(f"Post-period obs: {(df['post'] == 1).sum()}")

# ── 3. Stationarity tests (pre-period only) ───────────────────────────────────
pre_series = df.loc[df["post"] == 0, "outcome"].values

adf_stat, adf_p, *_ = adfuller(pre_series, autolag="AIC")
kpss_stat, kpss_p, *_ = kpss(pre_series, regression="c", nlags="auto")

print("\n── Stationarity ──────────────────────────────────────────────────────")
print(f"ADF  p = {adf_p:.4f}  {'✅ stationary' if adf_p < 0.05 else '⚠️  unit root possible'}")
print(f"KPSS p = {kpss_p:.4f}  {'✅ stationary' if kpss_p > 0.05 else '⚠️  non-stationary'}")

# ── 4. Main ITS model (HAC standard errors for autocorrelation-robust CIs) ────
model = smf.ols("outcome ~ time + post + time_since", data=df).fit(
    cov_type="HAC", cov_kwds={"maxlags": 4}
)
print("\n── Main ITS Model ────────────────────────────────────────────────────")
print(model.summary())

# Key estimates
level_change  = model.params["post"]
slope_change  = model.params["time_since"]
level_ci      = model.conf_int().loc["post"]
slope_ci      = model.conf_int().loc["time_since"]

print(f"\n▶ Level change at intervention : {level_change:+.3f}  "
      f"(95% CI: [{level_ci[0]:+.3f}, {level_ci[1]:+.3f}])")
print(f"▶ Slope change per month       : {slope_change:+.3f}  "
      f"(95% CI: [{slope_ci[0]:+.3f}, {slope_ci[1]:+.3f}])")

# ── 5. Pre-period fit diagnostics ─────────────────────────────────────────────
pre_mask      = df["post"] == 0
pre_observed  = df.loc[pre_mask, "outcome"]
pre_fitted    = model.fittedvalues[pre_mask]
pre_residuals = pre_observed - pre_fitted

mape = np.mean(np.abs(pre_residuals / pre_observed)) * 100
print(f"\n── Pre-Period Model Fit ──────────────────────────────────────────────")
print(f"MAPE: {mape:.2f}%  "
      + ("✅ good" if mape < 5 else "⚠️ acceptable" if mape < 10 else "🚨 poor"))

lb = acorr_ljungbox(pre_residuals, lags=10, return_df=True)
print(f"Ljung-Box (lag 10) p = {lb['lb_pvalue'].iloc[-1]:.4f}  "
      + ("✅ no autocorrelation" if lb['lb_pvalue'].iloc[-1] > 0.05
         else "⚠️ residual autocorrelation detected"))

sw_stat, sw_p = shapiro(pre_residuals)
print(f"Shapiro-Wilk p = {sw_p:.4f}  "
      + ("✅ residuals approx. normal" if sw_p > 0.05 else "⚠️ non-normal residuals"))

# ── 6. Counterfactual plot ────────────────────────────────────────────────────
# Counterfactual: post-period predictions WITHOUT the intervention
#   (set post=0 and time_since=0 for all post-period rows)
cf_data = df.copy()
cf_data["post"] = 0
cf_data["time_since"] = 0
df["counterfactual"] = model.predict(cf_data)

fig, ax = plt.subplots(figsize=(12, 5))
ax.plot(df["time"], df["outcome"],      "o-", color="steelblue",  label="Observed",       lw=1.5, ms=4)
ax.plot(df["time"], model.fittedvalues, "-",  color="steelblue",  label="Fitted",         lw=1, linestyle="--", alpha=0.6)
ax.plot(df["time"], df["counterfactual"], "-", color="tomato",    label="Counterfactual", lw=2, linestyle="--")
ax.axvline(x=INTERVENTION_TIME + 0.5, color="black", linestyle=":", lw=1.5, label="Intervention (month 37)")
ax.fill_between(
    df.loc[df["post"] == 1, "time"],
    df.loc[df["post"] == 1, "counterfactual"],
    df.loc[df["post"] == 1, "outcome"],
    alpha=0.15, color="green", label="Estimated effect"
)
ax.set(xlabel="Month", ylabel="Outcome", title="Interrupted Time Series — Observed vs. Counterfactual")
ax.legend()
plt.tight_layout()
plt.savefig("its_main.png", dpi=150)
plt.show()
print("Plot saved → its_main.png")
```

## Stage 4: Robustness Checks

Two checks are most valuable here.

### Check A — Placebo intervention date

Run the exact same model but pretend the intervention happened at month 18 (midpoint of your pre-period). If you find a significant "effect," it suggests the model is picking up trend noise or a pre-period structural break — not your policy.

```python
# ── Placebo intervention test ──────────────────────────────────────────────
PLACEBO_TIME = 18

df_placebo = df.loc[df["post"] == 0].copy()   # pre-period only
df_placebo["post_placebo"]       = (df_placebo["time"] > PLACEBO_TIME).astype(int)
df_placebo["time_since_placebo"] = np.maximum(0, df_placebo["time"] - PLACEBO_TIME)

model_placebo = smf.ols(
    "outcome ~ time + post_placebo + time_since_placebo",
    data=df_placebo
).fit(cov_type="HAC", cov_kwds={"maxlags": 4})

placebo_level = model_placebo.params["post_placebo"]
placebo_ci    = model_placebo.conf_int().loc["post_placebo"]

print(f"Placebo level change : {placebo_level:+.3f}  "
      f"(95% CI: [{placebo_ci[0]:+.3f}, {placebo_ci[1]:+.3f}])")
print(f"Placebo p-value      : {model_placebo.pvalues['post_placebo']:.4f}")
print("\n✅ Good: CI includes zero → no spurious effect at placebo date"
      if placebo_ci[0] < 0 < placebo_ci[1]
      else "⚠️ Concern: significant effect at placebo date — investigate pre-period stability")
```

### Check B — Sensitivity to pre-period length

Re-run the model using only months 13–60 (dropping the first year). If the estimated level change shifts substantially, the result may be sensitive to early pre-period patterns.

```python
      else "⚠️ Sensitive to pre-period length — interpret with caution")
```
